### Current Work

**Runtime Governance for Autonomous AI Coding Agents** — *Master's Thesis, Metosin*

A control-plane-style framework that makes the behaviour of AI coding agents
(Claude Code and Gemini CLI) observable and subject to runtime policy, without
modifying the agents themselves. It captures lifecycle events through native
hooks, routes them through a durable Temporal workflow, and delivers structured
telemetry to a self-hosted Langfuse platform. Governance layers handle policy
enforcement, anomaly detection, goal assessment, hallucination detection, and
data lineage.

<sub>Go · Python · Temporal · Langfuse · Docker · ClickHouse · MCP</sub>
